syntax $('.class')
example:
$('.small') all elements with class small
$('.small,.big') all elements with class big
$('div .small,.big') all div elements with class small and any element with class big
$('.small').css('border', '5px solid red') search all element with class small and set border as 5px solid red

$('.small .big') all elements with class big which is descendent of class small
OR
$('.small').filter('.big').css('border', '5px solid red') this is slower as it first searches all small and then
remove those who do not have big
